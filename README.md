# Working with Microservices in Go (Golang)

This project is based on the Udemy course: [Working with Microservices in Go (Golang)](https://www.udemy.com/course/working-with-microservices-in-go/)

## About the Course

Learn to build highly available, scalable, and resilient distributed applications using Go. This course covers:

- Microservice architecture fundamentals
- Building and connecting multiple Go microservices
- REST API, RPC, gRPC, and AMQP (RabbitMQ) communication
- Services include:
  - Front End (web pages)
  - Authentication (with Postgres)
  - Logging (with MongoDB)
  - Listener (RabbitMQ consumer)
  - Broker (entry point)
  - Mail (email sender)
- Deploying to Docker Swarm and Kubernetes
- Scaling and updating services with minimal downtime
